# Quick Math

Quick Math is a browser-based arithmetic quiz that lets you choose operations (add, subtract, multiply), answer a short set of generated questions, and see accuracy plus time per answer.

## Tech Stack
- React + TypeScript
- Vite tooling
- React Router for page flows
- CSS Modules for styling
- localStorage for persisted preferences

## Features
- Choose which operators are included via toggleable buttons
- Generate 7â€“10 random arithmetic questions with previous/current/next context
- Keyboard-friendly workflow (Enter to start/submit, Escape to restart)
- Live timer and per-question time tracking
- Results view with correctness icons and success ratio

## Run Locally
1) Install dependencies: `npm install`
2) Start dev server: `npm run dev`
3) Build for production: `npm run build`
